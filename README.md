# Best Buy 2.0

Python store simulation that models products, promotions, and store behavior. The project includes unit tests and separates business logic across focused Python modules.

## Features

- Product model with stock and pricing behavior
- Store logic for managing products
- Promotion system
- Command-line interaction through `main.py`
- Unit tests for product behavior

## Tech Stack

Python · Pytest · Object-Oriented Programming

## Project Structure

```text
Best-Buy-2.0/
├── main.py
├── products.py
├── promotions.py
├── store.py
└── test_product.py
```

## What This Project Demonstrates

- Object-oriented Python design
- Business logic separation
- Testing with Pytest
- Command-line application structure
